<h3>:fire: Technologies i used</h3>
<ul>
  <li>Kotlin</li>
  <li>MVVM Architecture</li>
  <li>Dagger Hilt</li>
  <li>Data Binding</li>
  <li>View Binding</li>
  <li>StateFlow</li>
  <li>Offline caching</li>
  <li>Motion Layout</li>
  <li>Datastore Preferences</li>
  <li>Kotlin Coroutines</li>
